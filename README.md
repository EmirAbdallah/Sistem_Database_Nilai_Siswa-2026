SISTEM DATABASE NILAI SISWA

Code ini berupa mini-aplikasi untuk menampilkan, menambah, memperbarui, dan menghapus data pada database nilai siswa.
Tujuan nya adalah agar memudahkan operator dalam penyusunan maupun pengolahan data-data siswa dalam bentuk yang rapi dan terstruktur.
